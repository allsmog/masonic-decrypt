How it works: The Spartans wrapped a leather strip around a wooden rod called a scytale. The message was written horizontally across the wrapped strip. When unwrapped, the letters appeared scrambled. Only someone with a rod of the same diameter could read the message by re-wrapping the strip.

Try it: Change the rod diameter to see how the cipher changes!
